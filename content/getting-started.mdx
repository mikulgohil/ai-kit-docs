# Getting Started

This guide walks you through installing AI Kit, running it on your project, and understanding what it generates.

## Prerequisites

- **Node.js 18+** вЂ” check with `node --version`
- **A project with `package.json`** вЂ” AI Kit reads this to detect your stack
- **Claude Code or Cursor** вЂ” at least one AI tool installed

## Step 1: Run AI Kit

Navigate to your project root and run:

```bash
npx @mikulgohil/ai-kit init
```

No global install needed вЂ” `npx` downloads and runs it directly.

AI Kit scans your project and shows what it detected:

## Step 3: Choose Your Tools

```

```

Select which AI tools your team uses. AI Kit generates configs for each.

## Step 4: Handle Conflicts

If you already have a `CLAUDE.md` or `.cursorrules`:

```

вќЇ Overwrite вЂ” replace with fresh generated files
  Skip вЂ” keep existing files, only add missing ones
```

## Step 5: Choose Hook Profile

```
? Which hook profile would you like?
вќЇ Standard вЂ” auto-format + typecheck + console.log warnings
  Minimal вЂ” auto-format + git push safety only
  Strict вЂ” standard + ESLint + stop-time audit
```

Hooks run automatically as you code. See [Hooks](/docs/hooks) for details on each profile.

## Step 7: Start Using AI

Open your project in Claude Code or Cursor. The AI assistant will automatically read the generated config files and follow your project's conventions.

Try a skill or slash command to verify:

```
/prompt-help
```

Or just describe what you want вЂ” the AI will auto-apply the right skill:

```
I'm not sure how to phrase this request, can you help me?
```

This launches the prompt-help workflow вЂ” a great way to verify everything is working.
